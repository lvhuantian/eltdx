# v1.0.1

## 修复

- 修正 `0x054c` 批量快照盘口解析：当前实盘只稳定确认买一 / 卖一，不再把尾部扩展区误解析成买二到买五。
- `client.get_quote()` 保持旧入口返回完整五档盘口的兼容行为，但内部改为用 `0x0547` 首次刷新补齐深度数据。

## 新增

- 新增 `client.get_quote_depth()` / `client.quotes.get_depth()`，用于直接通过 `0x0547` 查询买一到买五 / 卖一到卖五。

## 改进

- `eltdx-smoke` 使用真实股票样本做报价检查，避免首个样本为分类代码且价格为 0。
- MCP 依赖改为可选 extra：需要 MCP 服务时安装 `eltdx[mcp]`。
- 文档补充源码开发安装方式、盘口接口口径和 MCP 可选依赖说明。
