# 竞价数据

## 作用

把集合竞价过程明细和 09:25 最终成交快照放到一个结果里，方便同时看竞价过程和开盘结果。

| 项目   | 内容                                                                                                                                    |
| ---- | ------------------------------------------------------------------------------------------------------------------------------------- |
| 调用方法 | `client.helpers.auction_data(code, date)`                                                                                             |
| 返回模型 | `AuctionData`                                                                                                                         |
| 底层能力 | [`0x056a` 集合竞价明细](../methods/7709-集合竞价明细.md)、[`09:25 竞价成交快照`](../methods/7709-0925竞价成交快照.md)、[`0x054c` 批量快照](../methods/7709-批量快照.md) |

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    data = client.helpers.auction_data("sz000001", "2026-05-20")

print(data.open_price, data.open_change_pct, data.open_volume, data.open_amount)

if data.series:
    for point in data.series.points[:5]:
        print(point.time_label, point.price, point.matched_volume, point.unmatched_volume)
```

## 参数

| 参数                 | 含义                         |
| ------------------ | -------------------------- |
| `code`             | 股票代码                       |
| `date`             | 交易日；用于扫描 09:25 成交快照        |
| `include_series`   | 是否请求集合竞价明细，默认 `True`       |
| `include_snapshot` | 是否请求 09:25 快照，默认 `True`    |
| `include_quote`    | 是否请求行情快照补昨收和开盘金额，默认 `True` |
| `pre_close_price`  | 手动传昨收；历史日期要计算开盘涨幅时建议传这个值   |

## 返回字段

| 字段                | 含义                                           |
| ----------------- | -------------------------------------------- |
| `code`            | 完整代码                                         |
| `trading_date`    | 交易日                                          |
| `series`          | `AuctionSeries`，当前交易日集合竞价过程明细                |
| `snapshot_0925`   | `Auction0925Result`，09:25 最终成交快照             |
| `pre_close_price` | 昨收                                           |
| `open_price`      | 开盘价，优先来自 09:25 快照                            |
| `open_volume`     | 09:25 成交量，单位手                                |
| `open_amount`     | 09:25 成交额                                    |
| `open_change_pct` | 开盘涨幅，按 `open_price / pre_close_price - 1` 计算 |

## 两类数据怎么用

| 数据              | 看什么                            |
| --------------- | ------------------------------ |
| `series.points` | 9:15 到 9:25 之间价格、虚拟成交量、未匹配量的变化 |
| `snapshot_0925` | 9:25 的最终成交价、成交量、成交额            |

开盘涨幅和开盘金额通常看 `snapshot_0925` 更直接；竞价强弱过程可以看 `series.points`。

`0x056a` 集合竞价明细按当前交易日读取。`date` 不是今天时，Helper 会返回对应日期的 `snapshot_0925`，`series` 为空；历史日期不会用当前行情快照的昨收去计算开盘涨幅，需要计算时传 `pre_close_price`。
