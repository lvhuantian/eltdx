# 批量行情表

## 作用

给一批股票整理行情表，只合并代码表和行情快照，不请求财务基础信息。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.helpers.quote_table(codes)` |
| 返回模型 | `StockProfileTable` |
| 底层能力 | [`0x054c` 批量快照](../methods/7709-批量快照.md)、[`0x044d` 代码表](../methods/7709-代码表.md) |

## 示例

```python
from eltdx import TdxClient

codes = ["sz000001", "sh600000", "bj920971"]

with TdxClient(timeout=3) as client:
    table = client.helpers.quote_table(codes)

for row in table.rows:
    print(row.full_code, row.name, row.last_price, row.change_pct, row.amount)
```

## 返回字段

`quote_table()` 返回的也是 `StockProfileTable`，但财务相关字段通常为空。

| 字段 | 含义 |
| --- | --- |
| `full_code` / `name` | 完整代码 / 名称 |
| `category` / `board` | 品种分类 / 板块分类 |
| `last_price` / `pre_close_price` | 最新价 / 昨收 |
| `open_price` / `high_price` / `low_price` | 今开 / 最高 / 最低 |
| `change` / `change_pct` | 涨跌额 / 涨跌幅 |
| `volume_hand` | 成交量，单位手 |
| `amount` | 成交额 |
| `open_amount_yuan` | 开盘金额 |

## 和股票信息汇总的区别

| 方法 | 请求内容 |
| --- | --- |
| `quote_table()` | 行情快照 + 代码表 |
| `stock_profile_table()` | 行情快照 + 代码表 + 财务基础信息 |

如果只是做自选股行情列表，用 `quote_table()` 更轻。
