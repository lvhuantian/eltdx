# 概念板块成分股

## 作用

查询某个概念 / 题材里的股票，并按服务端返回的排名字段整理成表。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.helpers.topic_stocks(seed_code, topic_id=...)` |
| 返回模型 | `TopicStockTable` |
| 底层能力 | [`F10-题材内对比`](../methods/F10-题材内对比.md)、[`F10-热点题材`](../methods/F10-热点题材.md) |

## 示例

```python
from eltdx import TdxClient

client = TdxClient(timeout=3)

table = client.helpers.topic_stocks(
    "000034",
    topic_name="存储芯片",
    sort_by="zdf",
)

for row in table.rows[:10]:
    print(row.rank, row.full_code, row.name, row.change_pct)
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `seed_code` | 种子股票。服务端题材对比接口需要一个股票代码配合题材 ID 使用 |
| `topic_id` | 题材 ID，来自 `stock_topics()` / `client.f10.topic_ids()` / `client.f10.hot_topics()` |
| `topic_name` | 题材名称；不传 `topic_id` 时，Helper 会先在该股票的题材里按名称匹配 |
| `sort_by` | 排序字段，默认 `zdf`，表示按涨跌幅 |
| `section` | F10 对比分区，默认 `gndbzfsj` |

## 返回字段

`TopicStockTable`

| 字段 | 含义 |
| --- | --- |
| `seed_code` | 查询时使用的种子股票 |
| `topic_id` / `topic_name` | 题材 ID / 题材名称 |
| `sort_by` | 排序字段 |
| `rows` | `TopicStock` 列表 |
| `count` | 股票数量 |

`TopicStock`

| 字段 | 含义 |
| --- | --- |
| `rank` | 题材内排名 |
| `full_code` / `exchange` / `market_id` / `code` | 股票代码和市场 |
| `name` | 股票简称 |
| `change_pct` | 当日涨跌幅 |
| `change_pct_3d` / `change_pct_5d` | 近 3 日 / 近 5 日涨跌幅 |
| `change_pct_20d` / `change_pct_60d` | 近 20 日 / 近 60 日涨跌幅 |
| `change_pct_ytd` | 年初以来涨跌幅 |
| `trading_date` | 统计日期 |
| `raw` | F10 原始行 |
