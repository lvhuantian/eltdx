# 7709-换手率

## 作用

用成交量和指定日期股本本地计算换手率。服务器不直接返回这个结果。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.get_turnover(code, volume, on=None, unit="hand")` |
| 数据来源 | [`0x000f`](../COMMANDS_7709.md#cmd-0x000f) 的流通股本 |
| 返回 | `float` |

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    turnover = client.get_turnover("sz000001", 123456, on="2026-05-20", unit="hand")
    print(turnover)
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `volume` | 成交量 |
| `on` | 使用哪天之前最近一次流通股本 |
| `unit` | `hand` 表示手，`share` 表示股 |
| `refresh` | 是否跳过股本变迁缓存 |

`unit` 还支持 `hands`、`lot`、`lots`、`shares`、`stock` 这些英文别名。

## 计算

```text
换手率 = 成交股数 / 流通股本 * 100
```

`unit="hand"` 时，传入成交量会先乘以 100 转成股。

## 数据来源

| 项目 | 来源 |
| --- | --- |
| 成交量 | 调用者自己传入，可以来自 K 线、分时或行情快照 |
| 流通股本 | `client.get_equity(code, on=...)` |
| 股本底层接口 | [7709-股本变迁GBBQ.md](7709-股本变迁GBBQ.md) |

如果指定日期之前找不到流通股本，返回 `0.0`。
