# F10-沪深股通持仓

## 作用

查询沪股通 / 深股通持仓变化，包括持股比例、持股数量、变动股数、收盘价等。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.northbound_holding(code, section="bszj", page=1, page_size=20)` |
| 底层 Entry | [`CWServ.tdxf10_gg_zlcc`](../F10_7615.md#entry-cwserv-tdxf10-gg-zlcc) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
response = f10.northbound_holding("000034")

print(response.rows[:5])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `section` | 分区；默认 `bszj` |
| `filter_value` | 服务端筛选参数，默认空 |
| `page` / `page_size` | 分页参数 |

## 请求对照

```python
Entry = "CWServ.tdxf10_gg_zlcc"
body = {"Params": ["000034", "bszj", "", "1", "1", "20"]}
```

## 返回示例

```python
response.rows[0]
# {
#     "N001": "2026-03-31",
#     "N002": 0.63,
#     "N003": 4604186,
#     "N004": -1960712,
#     "N005": -29.8666,
#     "N006": 33.37
# }
```

## 字段对照

| 原生字段 | 中文含义 |
| --- | --- |
| `N001` | 指标日期 |
| `N002` | 持股比例% |
| `N003` | 持股数量，单位股 |
| `N004` | 变动股数，单位股 |
| `N005` | 较上期变化% |
| `N006` | 收盘价，单位元 |

部分页面会先用 `checkhsgbd` 判断沪深股通类型，常见返回字段 `hsg`：`1` 深股通、`2` 沪股通。当前封装的 `northbound_holding()` 默认直接取 `bszj` 持股变化序列。
