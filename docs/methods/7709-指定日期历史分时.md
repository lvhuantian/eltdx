# 7709-指定日期历史分时

## 作用

按日期查询某天的历史分时价格和分钟成交量，适合补单日历史分时。

| 项目 | 内容 |
| --- | --- |
| 主要调用 | `client.minutes.history(code, date)` |
| 旧版兼容 | `client.get_minute(code, date)` |
| 旧版兼容 · 别名 | `client.get_history_minute(code, date)` |
| 底层接口 | [`0x0fb4`](../COMMANDS_7709.md#cmd-0x0fb4) |
| 返回模型 | `MinuteSeries` |

这个接口按明确日期查询。另一个“近期历史分时”接口覆盖窗口更短，但会多返回今开和均价字段。

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    series = client.minutes.history("sz000001", "2026-05-20")
    series2 = client.get_history_minute("sz000001", "2026-05-20")
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `trading_date` / `date` | 交易日，支持 `YYYY-MM-DD`、`YYYYMMDD`、`date` |
| `include_raw` | 是否保留原始 payload |

## 解析字段

| `MinuteSeries` 字段 | 含义 |
| --- | --- |
| `exchange` / `market_id` / `code` / `full_code` | 市场和代码 |
| `trading_date` | 交易日 |
| `prev_close` | 昨收 |
| `points` | 分时点列表 |
| `count` | 分时点数量 |
| `volume_sum` | 分时成交量合计 |
| `raw_payload` | 原始 payload |

| `MinutePoint` 字段 | 含义 |
| --- | --- |
| `index` | 分时序号 |
| `time_label` / `time` | 时间文本 / 带日期时间 |
| `price` / `price_milli` | 当前价格 / 毫厘价 |
| `volume` | 该分钟成交量，单位手 |
| `price_delta_raw` / `aux_delta_raw` | 协议差分原始字段 |
| `price_raw` | 价格差分累计后的原始价格口径 |
| `record_hex` | 单条原始十六进制 |

## 使用建议

| 场景 | 建议 |
| --- | --- |
| 要指定很早的某一天 | 优先用这个接口 |
| 只查近期交易日且想要均价 / 今开 | 可以用 [7709-近期历史分时.md](7709-近期历史分时.md) |
| 不传日期查今天 | 用 [7709-当日分时.md](7709-当日分时.md) |
