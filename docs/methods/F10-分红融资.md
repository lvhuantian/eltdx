# F10-分红融资

## 作用

查询分红、融资相关数据。默认 `section="fh"` 返回分红方案历史。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.dividend_financing(code, section="fh")` |
| 底层 Entry | [`CWServ.tdxf10_gg_fhrz`](../F10_7615.md#entry-cwserv-tdxf10-gg-fhrz) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
response = f10.dividend_financing("000034")

print(response.rows[:5])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `section` | 分区；`fh` 常用于分红方案历史 |

## 请求对照

```python
Entry = "CWServ.tdxf10_gg_fhrz"
body = {"Params": ["000034", "fh"]}
```

## 字段对照

`section="fh"` 返回分红方案历史，常用字段如下：

| 原生字段 | 中文含义 |
| --- | --- |
| `rq` | 分红年度 / 报告期 |
| `T003` | 董事会日期 |
| `T004` | 分红方案 |
| `T006` | 每股收益 |
| `T026` | 加权净资产收益率% |
| `T021` | 股权登记日 |
| `T023` | 除权派息日 |
| `T036` | 方案进度 |
| `aT036` | 方案进度码 |
| `glzfl` | 股利支付率% |
| `jdcode` | 派发对象 |

这个 Entry 还有多个分区，例如 `zf` 增发、`pf` 配股、`gqjl` 股权激励、`fhpm_gxl` 股息率排名、`fhlszs_gxl` 股息率历史走势。封装方法默认查 `fh`，其他分区可以通过 `section` 传入。

如果需要除权除息事件的行情协议字段，对应 7709 的股本变迁接口。
