# 7709-除权除息整理

## 作用

从 `0x000f` 股本变迁里筛出除权除息事件，并整理成分红、配股价、送转股、配股字段。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.get_xdxr(code)` |
| 数据来源 | [`0x000f`](../COMMANDS_7709.md#cmd-0x000f) |
| 返回模型 | `list[XdxrRecord]` |

这是 SDK 本地整理方法，数据来自 GBBQ。它只保留 `category_raw == 1` 的除权除息记录。

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    records = client.get_xdxr("sz000001")
    for item in records:
        print(item.date, item.fenhong, item.songzhuangu)
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `refresh` | 是否跳过股本变迁缓存 |

## 解析字段

| `XdxrRecord` 字段 | 含义 |
| --- | --- |
| `code` | 完整代码 |
| `date` / `time` | 日期 / 派生时间 |
| `category` / `category_name` | 类别编号 / 类别名称 |
| `fenhong` | 分红 |
| `peigujia` | 配股价 |
| `songzhuangu` | 送转股 |
| `peigu` | 配股 |

## 来源关系

| 字段 | 来源 |
| --- | --- |
| `fenhong` | `CapitalChangeRecord.c1_value` |
| `peigujia` | `CapitalChangeRecord.c2_value` |
| `songzhuangu` | `CapitalChangeRecord.c3_value` |
| `peigu` | `CapitalChangeRecord.c4_value` |

这些字段主要给本地复权因子、除权除息表和旧版兼容调用使用。普通取复权 K 线可以直接用 `client.get_kline(..., adjust="qfq")`。
