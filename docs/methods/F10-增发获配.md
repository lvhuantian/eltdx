# F10-增发获配

## 作用

先查询增发获配可选日期，再按日期查询获配机构明细。

| 项目 | 内容 |
| --- | --- |
| 日期列表 | `client.f10.allotment_dates(code)` |
| 获配明细 | `client.f10.allotment_details(code, date)` |
| 底层 Entry | [`CWServ.tdxf10_gg_fhrz_zfhpmx`](../F10_7615.md#entry-cwserv-tdxf10-gg-fhrz-zfhpmx) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
dates = f10.allotment_dates("000034")

first_date = dates.rows[0]["rq"]
details = f10.allotment_details("000034", first_date)

print(dates.rows[:3])
print(details.rows[:5])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `date` | 增发日期，通常来自 `allotment_dates()` 返回行 |

## 请求对照

```python
f10.allotment_dates("000034")
# {"Params": ["zfpg_bgq", "000034", ""]}

f10.allotment_details("000034", "20250101")
# {"Params": ["zfpg", "000034", "20250101"]}
```

## 返回示例

```python
dates.rows[0]
# {"rq": "2016-03-01"}

details.rows[0]
# {
#     "rq": "2016-03-01",
#     "T004": "获配机构名称",
#     "T007": 1000000,
#     "T008": 1000000,
#     "T012": 12345678.0,
#     "T009": 6,
#     "T006": "机构类型",
#     "jjrq": "2016-09-01",
#     "T002": "股东ID",
#     "T003": "详情参数",
#     "id": "标记"
# }
```

## 字段对照

| 原生字段 | 中文含义 |
| --- | --- |
| `rq` | 增发日期 |
| `T004` | 获配机构 |
| `T007` | 获配数量，单位股 |
| `T008` | 申购数量，单位股 |
| `T012` | 获配金额，单位元 |
| `T009` | 锁定期，单位月 |
| `T006` | 机构类型 |
| `jjrq` | 解禁日期 |
| `T002` | 股东 ID |
| `T003` | 通达信详情参数 |
| `id` | 股东进出详情标记 |

`allotment_dates()` 只返回可选日期；真正的获配机构、数量、金额要再调用 `allotment_details(code, date)`。
