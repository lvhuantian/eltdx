# F10-股东增减持

## 作用

查询股东增持、减持计划和相关公告信息。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.shareholder_change_plans(code, page=1, page_size=20)` |
| 底层 Entry | [`CWServ.tdxf10_gg_gdyj`](../F10_7615.md#entry-cwserv-tdxf10-gg-gdyj) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
response = f10.shareholder_change_plans("000034", page=1, page_size=20)

print(response.rows[:5])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `page` | 页码 |
| `page_size` | 每页条数 |
| `filter1` / `filter2` | 服务端筛选参数，默认空 |

## 请求对照

```python
Entry = "CWServ.tdxf10_gg_gdyj"
body = {"Params": ["000034", "gdzjcjh", "", "", "1", "1", "20"]}
```

## 字段对照

| 原生字段 | 中文含义 |
| --- | --- |
| `N001` | 最新公告日 |
| `N002` | 变动方向 |
| `N003` | 股东名称 |
| `N004` | 股东身份 / 职务 |
| `N005` | 拟变动数量上限，单位股 |
| `N006` | 拟变动上限占总股本比例% |
| `N007` | 保留字段 |
| `N008` | 拟变动资金上限，单位元 |
| `N009` | 起始变动日期 |
| `N010` | 截止变动日期 |
| `N011` | 进度 |
| `N012` | 详情记录 ID |

适合做股东行为、减持计划和公告跟踪。
