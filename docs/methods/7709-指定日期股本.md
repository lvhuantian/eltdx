# 7709-指定日期股本

## 作用

从股本变迁里整理股本变化记录，并取某日期之前最近一次流通股本和总股本。

| 项目 | 内容 |
| --- | --- |
| 变迁列表 | `client.get_equity_changes(code)` |
| 指定日期 | `client.get_equity(code, on=...)` |
| 数据来源 | [`0x000f`](../COMMANDS_7709.md#cmd-0x000f) |
| 返回模型 | `EquityResponse` / `EquityRecord | None` |

这是 SDK 本地整理方法，数据来自 GBBQ。它从 GBBQ 里筛出股本变化类事件。

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    changes = client.get_equity_changes("sz000001")
    equity = client.get_equity("sz000001", on="2026-05-20")
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `on` | 目标日期；取这个日期之前最近一条股本记录 |
| `refresh` | 是否跳过股本变迁缓存 |

## 解析字段

| `EquityResponse` 字段 | 含义 |
| --- | --- |
| `count` | 记录数 |
| `items` | 股本变化记录 |

| `EquityRecord` 字段 | 含义 |
| --- | --- |
| `code` | 完整代码 |
| `date` / `time` | 日期 / 派生时间 |
| `category` / `category_name` | 类别编号 / 类别名称 |
| `float_shares` | 流通股本，单位股 |
| `total_shares` | 总股本，单位股 |

## 筛选规则

`get_equity_changes()` 目前保留这些 GBBQ 类别：

| `category` | 类别名 |
| ---:| --- |
| `2` | 送配股上市 |
| `3` | 非流通股上市 |
| `5` | 股本变化 |
| `7` | 股份回购 |
| `8` | 增发新股上市 |
| `9` | 转配股上市 |
| `10` | 可转债上市 |

字段来源：

| `EquityRecord` 字段 | 来源 |
| --- | --- |
| `float_shares` | `CapitalChangeRecord.c3_value` |
| `total_shares` | `CapitalChangeRecord.c4_value` |

`get_equity(code, on=...)` 会按日期排序，取 `on` 当天或之前最近的一条股本记录；找不到时返回 `None`。
