# 7709-当日分时

## 作用

查询当前交易日分时序列，返回每分钟价格、成交量和均价等字段。

| 项目 | 内容 |
| --- | --- |
| 主要调用 | `client.minutes.today(code)` |
| 旧版兼容 | `client.get_minute(code)` |
| 底层接口 | [`0x0537`](../COMMANDS_7709.md#cmd-0x0537) |
| 返回模型 | `MinuteSeries` |

不传日期调用 `client.get_minute(code)` 时，走的就是这个当日分时接口。

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    series = client.minutes.today("sz000001")
    print(series.count, series.volume_sum)
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `include_raw` | 是否保留原始 payload |

## 解析字段

| `MinuteSeries` 字段 | 含义 |
| --- | --- |
| `exchange` | 市场前缀，`sz` / `sh` / `bj` |
| `market_id` | 市场编号，`0=sz`、`1=sh`、`2=bj` |
| `code` | 六位代码 |
| `full_code` | 完整代码 |
| `trading_date` | 交易日；当日分时响应本身不带日期，所以这里通常为空 |
| `reserved_zero` | 响应头保留字段 |
| `points` | 分时点列表 |
| `count` | 分时点数量 |
| `volume_sum` | 分时成交量合计 |
| `raw_payload` | 原始 payload |

| `MinutePoint` 字段 | 含义 |
| --- | --- |
| `index` | 分时序号 |
| `time_label` | 时间文本，例如 `09:30`、`10:01` |
| `time` | 带日期的时间；当日分时可能为空 |
| `price` / `price_milli` | 当前价格 / 千分价格单位 |
| `avg_price` | 均价 |
| `volume` | 该分钟成交量，单位手 |
| `price_field` / `avg_field` | 协议中的价格字段 / 均价字段 |
| `price_raw` / `avg_raw` | 还原后的原始价格口径 |
| `record_hex` | 单条原始十六进制 |

## 时间和价格口径

| 项目 | 说明 |
| --- | --- |
| 分钟序号 | `index=0` 对应开盘后第一个分时点 |
| 时间标签 | SDK 根据 A 股交易分钟序列生成 `time_label` |
| 日期 | 当日接口不返回日期；要带日期的历史分时用 `client.minutes.history()` 或 `client.minutes.recent()` |
| 价格 | SDK 已把协议差分还原成 `price` |
| 均价 | `avg_price` 是服务端返回的均价口径字段 |
