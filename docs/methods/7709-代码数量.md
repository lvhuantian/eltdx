# 7709-代码数量

## 作用

查询沪、深、北某个市场的证券数量，常用于分页拉代码表前确定规模。

| 项目 | 内容 |
| --- | --- |
| 主要调用 | `client.codes.count(market)` |
| 旧版兼容 | `client.get_count(market, refresh=False)` |
| 底层接口 | [`0x044e`](../COMMANDS_7709.md#cmd-0x044e) |
| 返回 | `int` |

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    sz_count = client.codes.count("sz")
    sh_count = client.get_count("sh")
```

配合代码表分页：

```python
count = client.get_count("sz")
codes = client.codes.list("sz", start=0, limit=min(count, 1600))
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `market` | 市场，支持 `sz`、`sh`、`bj`，也支持 `0`、`1`、`2` |
| `refresh` | 仅 `get_count()` 支持；`True` 表示跳过内存缓存重新请求 |

## 返回

| 返回值 | 含义 |
| --- | --- |
| `int` | 该市场代码数量 |

## 使用说明

| 项目 | 说明 |
| --- | --- |
| 底层请求 | `0x044e` 只传市场和客户端日期 |
| 返回内容 | 只返回数量，不返回具体代码 |
| 常用搭配 | 先取数量，再用 [7709-代码表.md](7709-代码表.md) 分页拉具体记录 |
| 缓存 | `client.get_count()` 默认缓存；`client.codes.count()` 每次直接请求 |
