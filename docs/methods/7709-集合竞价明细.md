# 7709-集合竞价明细

## 作用

查询当前交易日集合竞价阶段的价格、虚拟成交量、未匹配量等明细记录。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.auctions.series()` / `client.get_call_auction()` |
| 底层接口 | [`0x056a`](../COMMANDS_7709.md#cmd-0x056a) |
| 返回模型 | `AuctionSeries` |

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    series = client.auctions.series("sz000001")
    for point in series.points:
        print(point.time_label, point.price, point.matched_volume)
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `include_raw` | 是否保留原始 payload |

这个接口请求参数里没有交易日期字段，返回当前交易日集合竞价明细。历史某天 09:25 的最终竞价成交可以用 [7709-0925竞价成交快照.md](7709-0925竞价成交快照.md)。

## 解析字段

| `AuctionSeries` 字段 | 含义 |
| --- | --- |
| `exchange` / `market_id` / `code` / `full_code` | 市场和代码 |
| `mode_or_selector_raw` | 模式 / 选择器原始值 |
| `start_raw` | 起始参数原始值 |
| `limit_or_count_raw` | 数量参数原始值 |
| `points` | 竞价明细记录 |
| `count` | 记录数 |
| `raw_payload` | 原始 payload |

| `AuctionPoint` 字段 | 含义 |
| --- | --- |
| `index` | 序号 |
| `minute_of_day_raw` | 当日分钟原始值 |
| `second_raw` | 秒原始值 |
| `time_label` | 时间 |
| `time_seconds` | 当日秒数 |
| `price` / `price_milli` | 竞价价格 / 毫厘价 |
| `matched_volume` | 虚拟成交量，单位手 |
| `unmatched_signed_raw` | 未匹配量带符号原始值 |
| `unmatched_volume` | 未匹配量，单位手 |
| `unmatched_direction_raw` | 未匹配方向原始值 |
| `reserved_zero_0e` | 保留字段 |
| `record_hex` | 单条原始十六进制 |
| `matched_amount_estimated` | 估算成交额，`price * matched_volume * 100` |

## 和 09:25 竞价成交快照的区别

| 方法 | 数据来源 | 适合场景 |
| --- | --- | --- |
| `client.auctions.series()` | `0x056a` 集合竞价明细 | 看竞价阶段价格、匹配量、未匹配量的过程 |
| `client.get_auction_0925()` | 从 `0x0fc6` 历史成交明细扫描 | 获取某天 09:25 竞价成交快照 |
