# 7709-近期历史分时

## 作用

查询服务端近期窗口内的历史分时。它和 `0x0fb4` 都是历史分时，但覆盖范围以服务端近期窗口为准。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.minutes.recent()` |
| 底层接口 | [`0x0feb`](../COMMANDS_7709.md#cmd-0x0feb) |
| 返回模型 | `MinuteSeries` |

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    series = client.minutes.recent("sz000001", "2026-05-20")
```

不传日期时使用当前日期：

```python
today_like = client.minutes.recent("sz000001")
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `trading_date` | 近期窗口内的交易日；不传时使用当前日期 |
| `include_raw` | 是否保留原始 payload |

## 解析字段

| `MinuteSeries` 字段 | 含义 |
| --- | --- |
| `exchange` / `market_id` / `code` / `full_code` | 市场和代码 |
| `trading_date` | 交易日 |
| `prev_close` | 昨收 |
| `open_price` | 今开 |
| `date_selector_raw` | 服务端近期日期选择器原始值 |
| `points` | 分时点列表 |
| `count` | 分时点数量 |
| `volume_sum` | 分时成交量合计 |
| `raw_payload` | 原始 payload |

| `MinutePoint` 字段 | 含义 |
| --- | --- |
| `index` | 分时序号 |
| `time_label` / `time` | 时间文本 / 带日期时间 |
| `price` / `price_milli` | 分时价格 / 千分价格单位 |
| `avg_price` | 均价 |
| `volume` | 分钟成交量，单位手 |
| `price_field` / `avg_field` | 协议中的价格字段 / 均价字段 |
| `price_raw` / `avg_raw` | 还原后的原始价格口径 |
| `record_hex` | 单条原始十六进制 |

## 覆盖范围

`0x0feb` 是近期窗口接口，能查多久由服务端决定；需要查更早日期时，用 [7709-指定日期历史分时.md](7709-指定日期历史分时.md)。
