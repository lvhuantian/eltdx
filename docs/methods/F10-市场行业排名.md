# F10-市场行业排名

## 作用

查询市场 / 行业排名明细，包括当前股票排名、排名变化、同组股票代码、简称、市场和更新时间。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.ranking_detail(code, section="scpmdela")` |
| 底层 Entry | [`CWServ.tdxf10_gg_zxts_rqpm`](../F10_7615.md#entry-cwserv-tdxf10-gg-zxts-rqpm) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
response = f10.ranking_detail("000034")

print(response.rows[:5])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `section` | 排名分区；默认 `scpmdela` |

## 请求对照

```python
Entry = "CWServ.tdxf10_gg_zxts_rqpm"
body = {"Params": ["000034", "scpmdela"]}
```

## 返回示例

```python
response.tables[0].rows[0]
# {
#     "pm": "419",
#     "prepm": "53",
#     "zqdm": "000034",
#     "zqjc": "神州数码",
#     "sc": "0",
#     "modtime": "2026-05-28 21:10:00"
# }
```

## 返回表

| ResultSet | 含义 |
| --- | --- |
| `table0` | 当前股票自己的排名 |
| `table1` | 同组排名列表 |

## 字段对照

| 原生字段 | 中文含义 |
| --- | --- |
| `pm` | 排名 |
| `prepm` | 上期排名 / 排名变化参考 |
| `zqdm` / `gpdm` | 证券代码 |
| `zqjc` | 证券简称 |
| `sc` | 市场代码，通常 `0` 深市、`1` 沪市、`2` 北交所 |
| `modtime` | 更新时间 |

`section="scpmdela"` 常用于市场排名明细，`section="hypmdelat"` 常用于行业排名明细。
