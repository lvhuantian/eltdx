# F10-题材内对比

## 作用

查询某个题材内股票对比和排名。`topic_compare_first()` 会自动取股票第一个题材 ID 后再查询。

| 项目 | 内容 |
| --- | --- |
| 指定题材 | `client.f10.topic_compare(code, topic_id, ...)` |
| 首个题材 | `client.f10.topic_compare_first(code, ...)` |
| 底层 Entry | [`CWServ.tdxf10_gg_rdtc_gndb`](../F10_7615.md#entry-cwserv-tdxf10-gg-rdtc-gndb) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
response = f10.topic_compare("000034", topic_id="123456")
auto = f10.topic_compare_first("000034")

print(response.rows[:5])
print(auto.rows[:5])
```

## 从个股题材查题材内排名

常见用法是先拿某个股票的全部题材，再选一个题材 ID，最后查询这个题材里的个股排序。

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)

topics = f10.topic_ids("000034").rows
first_topic = topics[0]
topic_id = first_topic["t001"]
topic_name = first_topic["t002"]

ranking = f10.topic_compare("000034", topic_id=topic_id, sort_by="zdf")

print(topic_id, topic_name)
for row in ranking.rows[:10]:
    print(row["pm"], row["zqdm"], row["zqjc"], row["zdf"])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `topic_id` | 题材 ID，通常来自 `topic_ids()` 或 `hot_topics()` |
| `section` | 对比分区；默认 `gndbzfsj` |
| `sort_by` | 排序字段；默认 `zdf` |

## 请求对照

```python
Entry = "CWServ.tdxf10_gg_rdtc_gndb"
body = {"Params": ["gndbzfsj", "000034", "123456", "zdf"]}
```

`topic_compare_first("000034")` 会先请求：

```python
{"Params": ["rdtcgn", "000034"]}
```

## 字段对照

| 原生字段 | 中文含义 |
| --- | --- |
| `pm` | 题材内排名 |
| `zqdm` | 证券代码 |
| `zqjc` | 证券简称 |
| `sc` | 市场编号，通常 `0` 深市、`1` 沪市、`2` 北交所 |
| `zdf` | 当日涨跌幅 |
| `zdf_3d` | 近 3 日涨跌幅 |
| `zdf_5d` | 近 5 日涨跌幅 |
| `zdf_20d` | 近 20 日涨跌幅 |
| `zdf_60d` | 近 60 日涨跌幅 |
| `zdf_ys` | 年初以来涨跌幅 |
| `tjdate` | 统计日期 |

`sort_by="zdf"` 表示按涨跌幅排序。其他排序字段需要以通达信页面实际请求为准；没有确认前，建议先使用默认的 `zdf`。
