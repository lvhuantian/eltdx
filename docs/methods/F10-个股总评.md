# F10-个股总评

## 作用

查询个股总评，默认返回综合评分。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.stock_score(code, section="pf", arg="")` |
| 底层 Entry | [`CWServ.tdxf10_gg_ggzp`](../F10_7615.md#entry-cwserv-tdxf10-gg-ggzp) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
response = f10.stock_score("000034")

print(response.rows[:3])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `section` | 分区；`pf` 常用于综合评分 |
| `arg` | 服务端扩展参数，默认空 |

## 请求对照

```python
Entry = "CWServ.tdxf10_gg_ggzp"
body = {"Params": ["pf", "000034", "", ""]}
```

## 子功能

| `section` | 含义 |
| --- | --- |
| `pf` | 综合评分 |
| `wwdfx` | 五维度分析 |
| `jdzf_zxt` / `jdzf_bs` | 阶段涨幅和走势评价 |
| `ldts` / `fxts` / `znzd` | 亮点、风险、智能诊断 |
| `cmfb` | 筹码分布 |
| `rzrq` | 融资融券 |
| `jgpj` / `jgmbj` | 机构评级、目标价 |
| `jgcc_bg` / `jgcc_tb` | 机构持仓变更 / 同比 |

## 字段对照

`section="pf"` 综合评分常用字段：

| 原生字段 | 中文含义 |
| --- | --- |
| `N001` | 综合评分 / 总分 |
| `N002` | 行业排名名次 |
| `N003` | 行业排名总数 |
| `N004` | A 股市场排名名次 |
| `N005` | A 股市场排名总数 |
| `N006` | 打败 A 股百分比 |
| `N007` | 日期 |
| `N008..N011` | 资金面、基本面、消息面、主题面评分 |
| `N012` | 行业名 |
| `N018` | 股票名 |

其他分区字段会变。比如 `jgpj` 常见为日期和买入 / 增持 / 中性 / 减持 / 卖出家数，`jgmbj` 常见为日期、股价、目标价和上涨空间。
