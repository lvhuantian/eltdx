# F10-估值市场数据

## 作用

查询估值和市场数据，例如 PE(TTM)、PB(MRQ)、市销率、市现率、估值百分位、流通市值、总市值等。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.valuation(code, req_id="200191", page=0, page_size=20)` |
| 底层 Entry | [`HQServ.hq_nlp_gpsj`](../F10_7615.md#entry-hqserv-hq-nlp-gpsj) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
response = f10.valuation("000034")

print(response.rows[:5])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `req_id` | 估值请求类型；默认 `200191` |
| `page` / `page_size` | 分页参数 |
| `extra` | 额外覆盖请求字段 |

## 请求对照

```python
Entry = "HQServ.hq_nlp_gpsj"
body = [{
    "ReqId": "200191",
    "Code": "000034|0",
    "BeginDate": "0",
    "EndDate": "0",
    "Page": "0",
    "PageSize": "20",
    "modname": "mod_gpsj.dll",
}]
```

## ReqId 对照

| `req_id` | 含义 |
| --- | --- |
| `200191` | 估值表 |
| `200135` | 年度 / 月度涨跌幅 |
| `200192` | 单估值指标时间序列 |
| `200124` | PE/PB Band |

## 字段对照

`200191` 估值表通常 `table0` 是分页元信息，`table1` 是估值序列：

| 原生字段 | 中文含义 |
| --- | --- |
| `DATE` | 日期 |
| `PETTM` | PE(TTM) |
| `PEBFW` | PE 百分位 |
| `PBMRQ` | PB(MRQ) |
| `PBBFW` | PB 百分位 |
| `PCFOCFTTM` | 市现率(TTM) |
| `PCFBFW` | 市现率百分位 |
| `PSTTM` | 市销率(TTM) |
| `PSBFW` | 市销率百分位 |
| `PEG` | PEG |
| `AVGMVM` | 流通市值 |
| `ALIQMV` | 总市值 |

`200135` 返回年度 / 月度涨跌幅，`N001` 是年份，`N002..N013` 是 1-12 月涨跌幅，`N014` 是年度涨跌幅。`200192` 返回单估值指标时间序列，字段为 `DATE`、`value`、`bfw`。
