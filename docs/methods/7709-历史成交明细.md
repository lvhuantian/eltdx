# 7709-历史成交明细

## 作用

分页查询指定日期成交明细增强数据，包含价格、成交量、方向状态、委托笔数等字段。

| 项目 | 内容 |
| --- | --- |
| 主要调用 · 单页 | `client.trades.history(code, date, ...)` |
| 旧版兼容 · 单页 | `client.get_history_trade(code, date, ...)` |
| 旧版兼容 · 全日 | `client.get_history_trade_day(code, date)` |
| 底层接口 | [`0x0fc6`](../COMMANDS_7709.md#cmd-0x0fc6) |
| 返回模型 | `TradePage` |

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    page = client.trades.history("sz000001", "2026-05-20", start=0, count=2000)
    all_ticks = client.get_history_trade_day("sz000001", "2026-05-20")
```

`client.get_trades(code, date)` 传入日期时，也会走这个历史成交明细接口。

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `trading_date` / `date` | 交易日，支持 `YYYY-MM-DD`、`YYYYMMDD`、`date` |
| `start` | 起始位置，从 `0` 开始 |
| `count` | 本页条数，便捷方法默认 `2000` |
| `include_raw` | 是否保留原始 payload |

## 解析字段

| `TradePage` 字段 | 含义 |
| --- | --- |
| `exchange` / `market_id` / `code` / `full_code` | 市场和代码 |
| `start` / `request_count` | 请求起点 / 请求条数 |
| `trading_date` | 成交日期 |
| `price_base_raw_f32` | 价格基数原始字段 |
| `ticks` | 成交明细 |
| `count` | 实际成交条数 |
| `has_more` | `count >= request_count` 时为 `True`，表示可能还有下一页 |
| `raw_payload` | 原始 payload |

| `TradeTick` 字段 | 含义 |
| --- | --- |
| `index` | 本页内序号 |
| `absolute_index` | 全局序号 |
| `time_minutes` | 当日分钟数 |
| `time_label` / `trade_datetime` | 成交时间 |
| `price` / `price_milli` | 成交价 / 毫厘价 |
| `volume` | 成交量，单位手 |
| `order_count` | 该笔包含的订单数 |
| `status_raw` | 方向 / 状态原始值 |
| `side` | 方向，`buy`、`sell`、`neutral` 或状态名 |
| `price_delta_raw` | 成交价差分原始值 |
| `price_acc_raw` | 成交价差分累计值 |
| `reserved_zero` | 历史成交明细尾部保留字段 |
| `record_hex` | 单条原始十六进制 |
| `trade_amount_yuan` | 成交金额，`price * volume * 100` |

## 翻页规则

| 项目 | 说明 |
| --- | --- |
| 起始页 | `start=0` |
| 下一页 | `start += count` |
| 可能还有数据 | `page.has_more` 为 `True` |
| 拉某日全部成交明细 | 用 `client.get_history_trade_day(code, date)` |
