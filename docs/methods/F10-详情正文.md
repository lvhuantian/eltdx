# F10-详情正文

## 作用

按记录 ID 查询正文。常用于热点题材事件、违规处理、公司资讯等列表行的详情。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.detail(detail_type, record_id)` |
| 底层 Entry | [`CWServ.tdxf10_gg_idreq`](../F10_7615.md#entry-cwserv-tdxf10-gg-idreq) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
topics = f10.hot_topics("000034")

record_id = topics.rows[0]["arec"]
detail = f10.detail("sjqd", record_id)

print(detail.rows[:1])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `detail_type` | 详情类型，由来源列表决定 |
| `record_id` | 详情记录 ID |

## 请求对照

```python
Entry = "CWServ.tdxf10_gg_idreq"
body = {"Params": ["sjqd", "<上游列表返回的记录ID>"]}
```

## 返回示例

```python
response.rows[0]
# {
#     "Title": "标题 / 栏目名",
#     "Txt": "详情正文"
# }
```

## 字段对照

| 原生字段 | 中文含义 |
| --- | --- |
| `Title` | 标题 / 栏目名 |
| `Txt` | 详情正文 |

`detail_type` 和 `record_id` 通常来自上游列表行。热点题材明细的详情记录 ID 是 `arec`，实测常用 `detail_type="sjqd"` 查询事件正文；资本运作治理的详情记录 ID 常见是 `rec_id`。不同上游列表可能要求不同 `detail_type`，以抓到的来源页面请求为准。
