# F10-财务报表

## 作用

查询财务报表。默认 `report_type="zcfzb"` 返回资产负债表。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.finance_report(code, report_type="zcfzb")` |
| 底层 Entry | [`CWServ.tdxf10_gg_cwfx`](../F10_7615.md#entry-cwserv-tdxf10-gg-cwfx) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
balance = f10.finance_report("000034")
income = f10.finance_report("000034", report_type="lrb")

print(balance.rows[:3])
print(income.rows[:3])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `report_type` | 报表类型；当前确认 `zcfzb` 为资产负债表，其他类型返回情况以服务端为准 |

## 请求对照

```python
Entry = "CWServ.tdxf10_gg_cwfx"
body = {"Params": ["000034", "zcfzb", ""]}
```

## 字段对照

`report_type="zcfzb"` 返回资产负债表，`response.rows` 是多期报表行。常用字段如下：

| 类别 | 原生字段 | 中文含义 |
| --- | --- | --- |
| 报告期 | `rq` | 日期 / 报告期 |
| 资产 | `T007` | 货币资金 |
| 资产 | `T008` | 交易性金融资产 |
| 资产 | `T009` | 应收票据 |
| 资产 | `T010` | 应收账款 |
| 资产 | `T020` | 流动资产合计 |
| 资产 | `T039` | 资产总计 |
| 负债 | `T040` | 短期借款 |
| 负债 | `T042` | 应付票据 |
| 负债 | `T043` | 应付账款 |
| 负债 | `T053` | 流动负债合计 |
| 负债 | `T062` | 负债合计 |
| 权益 | `T063` | 实收资本 / 股本 |
| 权益 | `T064` | 资本公积金 |
| 权益 | `T067` | 未分配利润 |
| 权益 | `T077` | 归母权益合计 |
| 权益 | `T071` | 所有者权益合计 |
| 权益 | `T072` | 负债和股东权益合计 |

资产负债表响应里可能出现重复原生列名，例如 `T120`。`response.rows` 会保留为 `T120`、`T120__2`；需要严格按列位置处理时，用 `response.tables[0].row_cells`。

F10 财务报表是资料网关返回的表格数据；7709 的 `finance_batch()` 是行情协议里的基础财务字段，两者字段口径和数据来源不同。
