# F10-财务诊断

## 作用

查询财务诊断项，例如营运、盈利、成长、现金流、资产质量、Z 值预警和财务总评分。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.finance_diagnosis(code, section="yynl", scope="")` |
| 底层 Entry | [`CWServ.tdxf10_gg_cwzd`](../F10_7615.md#entry-cwserv-tdxf10-gg-cwzd) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
response = f10.finance_diagnosis("000034")

print(response.rows[:5])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `section` | 诊断分区；默认 `yynl` |
| `scope` | 服务端扩展筛选参数，默认空 |

## 请求对照

```python
Entry = "CWServ.tdxf10_gg_cwzd"
body = {"Params": ["yynl", "000034", ""]}
```

## 返回示例

```python
response.rows[0]
# {
#     "N000": "股票名称",
#     "N001": "对比行业",
#     "N002": "2025-12-31",
#     "N003": 12,
#     "N004": 3,
#     "N005": "等级码",
#     "N006": 78,
#     "N007": 85.5
# }
```

## 子功能

| `section` | 含义 |
| --- | --- |
| `yynl` | 营运能力 |
| `ylnl` | 盈利能力 |
| `cznl` / `cznl2` | 成长能力 |
| `xjl` | 现金流 |
| `zczl` | 资产质量 |
| `zzyj` | Z 值预警 |
| `cwzpf_zx` | 财务总评分最新 |
| `cwzpf_ls` | 财务总评分历史 |
| `ifcwzd` | 是否有财务诊断数据 |
| `cwldjfx` | 财务雷达 / 综合分析 |

## 字段对照

不同 `section` 的列会变化。`yynl` 这类能力诊断常见前段字段如下：

| 原生字段 | 中文含义 |
| --- | --- |
| `N000` | 股票名称 |
| `N001` | 对比行业 / 板块 |
| `N002` | 报告期 |
| `N003` | 个股排名 |
| `N004` | 排名变化 / 评分变化 |
| `N005` | 评价等级码 |
| `N006` | 综合评分 |
| `N007` | 分位 / 打败比例 |
| `N008..N024` | 指标组，按原生字段保留 |

`zzyj` 常见字段是 `N001` 报告期、`N002` Z 值 / 预警值、`N005..N010` 页面二次计算用财务指标。调用后如果 `rows` 为空，可以先打印 `response.tables[0].columns` 确认该股票该分区是否有数据。
