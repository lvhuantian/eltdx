# 7709-股本变迁GBBQ

## 作用

查询股本变迁 / 除权相关事件，包含除权除息、股本变化、增发、回购等记录。`get_gbbq()` 是兼容旧版的便捷名称。

| 项目 | 内容 |
| --- | --- |
| 主要调用 | `client.corporate.capital_changes(code)` |
| 旧版兼容 | `client.get_gbbq(code, ...)` |
| 底层接口 | [`0x000f`](../COMMANDS_7709.md#cmd-0x000f) |
| 返回模型 | `CapitalChangeBlock` |

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    block = client.corporate.capital_changes("sz000001")
    block2 = client.get_gbbq("sz000001")
```

遍历事件：

```python
for item in block.records:
    print(item.date, item.category_name, item.c1_value, item.c2_value, item.c3_value, item.c4_value)
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `include_raw` | 是否保留原始 payload |
| `refresh` | 仅 `get_gbbq()` 支持；跳过缓存 |

## 解析字段

| `CapitalChangeBlock` 字段 | 含义 |
| --- | --- |
| `exchange` / `market_id` / `code` / `full_code` | 市场和代码 |
| `block_count` | 服务端块数量 |
| `records` / `items` | 事件列表 |
| `count` | 实际记录数 |
| `raw_payload` | 原始 payload |

| `CapitalChangeRecord` 字段 | 含义 |
| --- | --- |
| `exchange` / `market_id` / `code` / `full_code` | 市场和代码 |
| `reserved_7` | 保留字节 |
| `date_raw` | 事件日期原始值，形如 `YYYYMMDD` |
| `date` | 事件日期 |
| `category_raw` / `category` | 事件类别编号 |
| `category_name` | 类别名称 |
| `c1_raw` / `c2_raw` / `c3_raw` / `c4_raw` | 四个原始字段 |
| `c1_float` / `c2_float` / `c3_float` / `c4_float` | 四个字段按 float32 解释的值 |
| `c1_value` / `c2_value` / `c3_value` / `c4_value` | 按类别解码后的四个业务值 |
| `c1` / `c2` / `c3` / `c4` | `c1_value` 到 `c4_value` 的简写属性 |
| `record_hex` | 单条原始十六进制 |
| `time` | 事件日期派生出的 `15:00` 时间 |

## 类别表

| `category_raw` | `category_name` | 参数口径 |
| ---:| --- | --- |
| `1` | 除权除息 | `c1..c4` 按 float32 解释，常用于分红、配股价、送转股、配股 |
| `2` | 送配股上市 | 股本数量类字段 |
| `3` | 非流通股上市 | 股本数量类字段 |
| `4` | 国家股配售 | 样本少，保留 raw，谨慎解释 |
| `5` | 股本变化 | 股本数量类字段，可用于整理流通股本 / 总股本 |
| `6` | 增发新股 | 股本数量类字段 |
| `7` | 股份回购 | 股本数量类字段 |
| `8` | 增发新股上市 | 股本数量类字段 |
| `9` | 转配股上市 | 股本数量类字段 |
| `10` | 可转债上市 | 股本数量类字段 |
| `11` | 扩缩股 | float32 比例类字段 |
| `12` | 非流通股缩股 | float32 比例类字段 |
| `13` | 送认购权证 | float32 权证类字段 |
| `14` | 送认沽权证 | float32 权证类字段 |
| `15` | 重整调整 | float32 比例类字段，样本里 `c3` 可和 F10 的转增显示对齐 |

## 单位口径

类别 `1/11/12/13/14/15` 按 float32 解释。其他类别按通达信数量编码解码后再乘 `10000`，因此 `c*_value` 是股数口径。

`get_gbbq()` 默认缓存结果；需要重拉时传 `refresh=True`。`include_raw=True` 时不走缓存，方便排查字段。
