# F10-公司资讯研报

## 作用

查询公司资讯，默认 `section="gsyj"` 返回研报列表，也可以按服务端分区查询监管措施等内容。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.company_news(code, section="gsyj", page=1, page_size=20)` |
| 底层 Entry | [`CWServ.tdxf10_gg_gszx`](../F10_7615.md#entry-cwserv-tdxf10-gg-gszx) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
reports = f10.company_news("000034")
supervision = f10.company_news("000034", section="jgcs")

print(reports.rows[:5])
print(supervision.rows[:5])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `section` | 分区；`gsyj` 常用于研报，`jgcs` 常用于监管措施 |
| `keyword` | 关键词 |
| `rating` | 评级筛选 |
| `page` / `page_size` | 分页参数 |

## 请求对照

```python
Entry = "CWServ.tdxf10_gg_gszx"
body = {"Params": ["000034", "gsyj", "", "0", "1", "20"]}
```

## 字段对照

| `section` | 含义 |
| --- | --- |
| `gsyj` | 公司研究 / 研报 |
| `jgcs` | 监管措施 / 处罚函件 |

`gsyj` 研报常用字段：

| 原生字段 | 中文含义 |
| --- | --- |
| `T004` | 评级类别 |
| `T009` | 研究员 |
| `T012` | 撰写日期 |
| `T011` | 研报详情 ID |
| `T039` | 研报标题 |
| `nflag` | 服务端标记 |
| `ybdz` | 研报地址 / 附件标识 |

`jgcs` 监管措施常用字段：`N001` 处罚公布日期、`N002` 处罚对象、`N003` 监管措施、`N004` 函件内容、`N005` 链接。

`rating` 评级筛选常用取值：`0` 全部、`5` 买入、`4` 增持、`3` 中性、`2` 减持、`1` 卖出。
