# 7709-财务基础信息

## 作用

批量查询基础财务字段，如流通股本、总股本、EPS、资产、负债、收入、利润等。

| 项目 | 内容 |
| --- | --- |
| 主要调用 | `client.corporate.finance_batch(codes)` |
| 缓存便捷调用 | `client.get_finance_batch(codes, refresh=False)` |
| 底层接口 | [`0x0010`](../COMMANDS_7709.md#cmd-0x0010) |
| 返回模型 | `FinanceBatch` |

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    batch = client.corporate.finance_batch(["sz000001", "sh600000"])
    selected = client.corporate.finance_batch(["sz000001"], fields=["流通股本", "total_shares"])
```

读取完整记录：

```python
record = batch.records[0]
print(record.full_code, record.circulating_shares, record.total_assets_yuan)
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `codes` | 单个代码或代码列表 |
| `fields` | 本地字段过滤；不改变服务端请求 |
| `include_raw` | 是否保留原始 payload |
| `refresh` | 仅 `get_finance_batch()` 支持；跳过缓存 |

## 派生字段

| 字段 | 计算方式 |
| --- | --- |
| `circulating_shares` | `liu_tong_gu_ben_raw_float * 10000` |
| `total_shares` | `zong_gu_ben_raw_float * 10000` |
| `total_assets_yuan` | `zong_zi_chan_raw_float * 1000` |
| `net_profit_yuan` | `jing_li_run_raw_float * 1000` |

## 字段过滤

`fields` 是 SDK 本地过滤，底层仍按 `0x0010` 请求完整记录。常用中文别名：

| 中文字段 | 实际属性 |
| --- | --- |
| `流通股本` | `circulating_shares` |
| `总股本` | `total_shares` |
| `总资产` | `total_assets_yuan` |
| `净利润` | `net_profit_yuan` |

示例返回：

```python
selected = client.corporate.finance_batch(["sz000001"], fields=["流通股本", "净利润"])
# [{"full_code": "sz000001", "流通股本": ..., "净利润": ...}]
```

## 单位口径

| 字段类型 | 换算 |
| --- | --- |
| 股本类 | raw 单位万股，实际股数 = raw * `10000` |
| 金额类 | raw 单位千元，人民币元 = raw * `1000` |
| 每股类 | raw 直接是元 |
| 日期类 | `YYYYMMDD` 转日期 |

地区和行业中文名当前没有直接暴露到 `FinanceRecord`，模型里保留的是 `province_raw` 和 `industry_raw`。
