# 7709-特殊品种涨跌停限制

## 作用

查询特殊品种涨跌停限制表。这个接口按表内位置分页；要查某个代码，可以先扫描后在本地建索引。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.limits.special()` / `client.limits.scan_special()` |
| 底层接口 | [`0x0452`](../COMMANDS_7709.md#cmd-0x0452) |
| 返回模型 | `SpecialLimitPage` / `list[SpecialLimitRecord]` |

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    page = client.limits.special(start_index=0)
    records = client.limits.scan_special()
    by_code = {item.full_code: item for item in records}
```

查询某个代码：

```python
records = client.limits.scan_special()
by_code = {item.full_code: item for item in records}
limit = by_code.get("sh688001")
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `start_index` | 从表内第几行开始 |
| `max_rows` | `scan_special()` 最多扫描多少行 |

## 解析字段

| `SpecialLimitPage` 字段 | 含义 |
| --- | --- |
| `start_index` | 本页起始行 |
| `records` | 记录列表 |
| `count` | 本页记录数 |
| `raw_payload` | 原始 payload |

| `SpecialLimitRecord` 字段 | 含义 |
| --- | --- |
| `exchange` | 市场前缀，`sz` / `sh` / `bj` |
| `market_id` | 市场编号，`0=sz`、`1=sh`、`2=bj` |
| `code_num` | 代码数字原始值 |
| `code` | 六位代码 |
| `full_code` | 完整代码 |
| `upper_price_raw_f32` | 涨停价原始 float32 |
| `lower_price_raw_f32` | 跌停价原始 float32 |
| `limit_up_price` | 涨停价 |
| `limit_down_price` | 跌停价 |
| `record_hex` | 单条原始十六进制 |

## 使用说明

| 项目 | 说明 |
| --- | --- |
| 查询方式 | 按表内 `start_index` 分页，不按代码直接点查 |
| 扫描方法 | `scan_special()` 会从起点连续翻页，直到返回空页或达到 `max_rows` |
| 适合用途 | 本地建立特殊品种涨跌停价索引 |
| 普通股票 | 不一定出现在这个表里，普通涨跌停规则通常按交易所规则计算 |
