# 7709-分类行情

## 作用

按分类分页查询行情列表，并让服务端按指定字段排序。适合做“沪深 A 股涨幅榜”“成交额榜”等列表。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.quotes.list_by_category(...)` |
| 底层接口 | [`0x054b`](../COMMANDS_7709.md#cmd-0x054b) |
| 返回模型 | `CategoryQuotePage` |

这个方法传“列表范围 + 排序字段 + 分页位置”。服务端会按条件返回这一页有哪些证券。

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    page = client.quotes.list_by_category("沪深A股", sort_by="涨幅", count=100)
    for item in page.records[:10]:
        print(item.full_code, item.last_price, item.change_pct)
```

按成交额取前 50 名：

```python
with TdxClient(timeout=3) as client:
    page = client.quotes.list_by_category("沪深A股", sort_by="成交额", count=50)
```

分页：

```python
first = client.quotes.list_by_category("沪深A股", sort_by="涨幅", start=0, count=80)
second = client.quotes.list_by_category("沪深A股", sort_by="涨幅", start=80, count=80)
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `category` | 分类编号或别名；常用 `"沪深A股"`、`"A股"`，也可以直接传数字分类 |
| `sort_by` | 排序字段名或排序编号；不传时按代码顺序 |
| `start` | 起始行，从 `0` 开始 |
| `count` | 本页条数 |
| `ascending` | 是否升序；字段排序默认降序 |

## 常用排序字段

| `sort_by` | 排序编号 | 说明 |
| --- | ---:| --- |
| `"代码"` | `0x0000` | 按列表默认代码顺序 |
| `"现价"` | `0x0006` | 按最新价 |
| `"成交额"` | `0x000a` | 按成交额 |
| `"涨幅"` | `0x000e` | 按 `change_pct` |
| `"封单额"` | `0x001c` | 按 `locked_amount` |
| `"开盘金额"` | `0x001d` | 按开盘金额 |
| `"涨速"` | `0x002e` | 按涨速 |
| `"短换手"` | `0x00cc` | 按短周期换手口径 |
| `"量涨速"` | `0x00d0` | 按量增速 |
| `"开盘抢筹"` | `0x010a` | 按开盘抢筹口径 |
| `"2分钟金额"` | `0x010c` | 按近 2 分钟金额 |
| `"开盘涨幅"` | `0x0119` | 按今开相对昨收涨幅 |
| `"最高涨幅"` | `0x011a` | 按最高价相对昨收涨幅 |
| `"最低涨幅"` | `0x011b` | 按最低价相对昨收涨幅 |
| `"回撤"` | `0x011e` | 按最高到现价的回撤 |
| `"攻击"` | `0x011f` | 按现价相对低点的攻击幅度 |

## 派生字段

| 字段 | 计算方式 |
| --- | --- |
| `change` | `last_price - pre_close_price` |
| `change_pct` | `change / pre_close_price * 100` |
| `locked_amount` | `bid1 * bid_vol1 * 100` |

协议文档里还确认过这些常见列表派生口径，当前模型没有单独做成属性时可以按下面公式自己算：

| 字段 | 计算方式 |
| --- | --- |
| `open_pct` | `(open_price - pre_close_price) / pre_close_price * 100` |
| `high_pct` | `(high_price - pre_close_price) / pre_close_price * 100` |
| `low_pct` | `(low_price - pre_close_price) / pre_close_price * 100` |
| `amplitude_pct` | `(high_price - low_price) / pre_close_price * 100` |
| `drawdown_pct` | `(high_price - last_price) / high_price * 100` |
| `attack_pct` | `(last_price - low_price) / pre_close_price * 100` |
| `entrust_ratio` | `(bid_vol1 - ask_vol1) / (bid_vol1 + ask_vol1) * 100` |
| `bid_ask_diff` | `bid_vol1 - ask_vol1` |

## 使用建议

| 场景 | 建议 |
| --- | --- |
| 做涨幅榜、成交额榜、量速榜 | 用这个方法，排序交给服务端 |
| 查指定几个代码 | 用 `client.get_quote()` |
| 列表第一页后继续翻页 | 下一页 `start += count` |
| 需要五档盘口完整买一到买五 | 用批量快照 `0x054c`；分类行情只返回一档盘口 |
