# 7709-小走势图

## 作用

查询单标的小型价格走势序列，适合列表页、概览页或轻量走势图。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.minutes.sparkline()` |
| 底层接口 | [`0x0fd1`](../COMMANDS_7709.md#cmd-0x0fd1) |
| 返回模型 | `SparklineSeries` |

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    series = client.minutes.sparkline("sz000001", selector=1, window=20)
    print(series.base_price, series.prices)
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `selector` | 走势选择器 |
| `window` | 窗口 / 数量参数 |
| `include_raw` | 是否保留原始 payload |

## 解析字段

| `SparklineSeries` 字段 | 含义 |
| --- | --- |
| `exchange` / `market_id` / `code` / `full_code` | 市场和代码 |
| `selector_raw` | 请求选择器 |
| `selector_echo` | 服务端回显的选择器 |
| `window_or_count_raw` | 请求窗口参数 |
| `max_count_raw` | 服务端返回最大数量口径 |
| `base_price` | 基准价 |
| `prices` | 价格序列 |
| `reserved_param_u32` | 保留参数原始值 |
| `count` | 实际价格点数量 |
| `raw_payload` | 原始 payload |

## 使用建议

这个方法返回的是轻量价格序列，字段比完整分时少。要画完整分时图，用 [7709-当日分时.md](7709-当日分时.md) 或 [7709-近期历史分时.md](7709-近期历史分时.md)。
