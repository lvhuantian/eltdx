# F10-题材概念行情

## 作用

查询题材概念行情、相关板块、板块成分股、主力控盘比例、主力资金走势和区间统计。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.theme_market(code, req_id="200743", page=-1, page_size=10)` |
| 底层 Entry | [`HQServ.hq_nlp_tcihq`](../F10_7615.md#entry-hqserv-hq-nlp-tcihq) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
response = f10.theme_market("000034", req_id="200743", page_size=-1)

print(response.rows[:5])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `req_id` | 题材行情请求类型；默认 `200743` |
| `page` / `page_size` | 分页参数 |
| `zq_num` | 部分 `ReqId` 使用的周期数量 |
| `extra` | 额外覆盖请求字段 |

## 请求对照

```python
Entry = "HQServ.hq_nlp_tcihq"
body = [{
    "ReqId": "200743",
    "setcode": 0,
    "code": "000034",
    "Page": -1,
    "PageSize": "-1",
    "modname": "mod_tcihq.dll",
}]
```

## ReqId 对照

| `req_id` | 含义 |
| --- | --- |
| `200741` | 区间统计 / 主题基础数据 |
| `200742` | 主力资金走势 |
| `200743` | 相关板块列表 |
| `200744` | 板块成分股 |
| `200745` | 每日主力控盘比例 |
| `200747` | 行业前十名股票主力控盘比例 |

## 字段对照

`200743` 相关板块通常返回两张表：`table0` 是元信息，`table1` 是板块列表。

| 原生字段 | 中文含义 |
| --- | --- |
| `table0.ans` | ReqId 回显 |
| `table0.date` | 数据日期 |
| `table0.time` | 数据时间 |
| `table0.total_num` | 总数量 |
| `table1.N001` | 板块市场 |
| `table1.N002` | 板块代码 |
| `table1.N003` | 板块名称 |
| `table1.N004` | 涨幅% |
| `table1.N005` | 涨停数 |

其他 `ReqId` 的字段会变化。常见口径：`200744` 返回板块成分股，`N001` 市场代码、`N002` 股票代码、`N003` 股票名称、`N004` 涨幅、`N005` 现价；`200745` 返回每日主力控盘比例，`N001` 日期、`N002` 主力控盘比例。
