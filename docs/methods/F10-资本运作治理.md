# F10-资本运作治理

## 作用

查询资本运作和公司治理数据，例如违规处理、担保明细、处罚公布日、案情进展、处罚决定、详情记录 ID。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.governance(code, section="wgcl", arg="")` |
| 底层 Entry | [`CWServ.tdxf10_gg_zbyz`](../F10_7615.md#entry-cwserv-tdxf10-gg-zbyz) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
violations = f10.governance("000034")
guarantees = f10.governance("000034", section="dbmx")

print(violations.rows[:5])
print(guarantees.rows[:5])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `section` | 分区；`wgcl` 常用于违规处理，`dbmx` 常用于担保明细 |
| `arg` | 服务端扩展参数，默认空 |

## 请求对照

```python
Entry = "CWServ.tdxf10_gg_zbyz"
body = {"Params": ["wgcl", "000034", ""]}
```

## 返回示例

```python
response.rows[0]
# {
#     "T003": "2025-07-03",
#     "T004": "案情进展",

```

## 字段对照

| `section` | 含义 |
| --- | --- |
| `wgcl` | 违规处理 |
| `dbmxbgq` | 担保明细可选报告期 |
| `dbmx` | 担保明细 |

`wgcl` 违规处理常用字段：

| 原生字段 | 中文含义 |
| --- | --- |
| `T003` | 立案日期 |
| `T004` | 案情进展 |
| `T007` | 处罚决定 |
| `T008` | 处罚公布日期 |
| `rec_id` | 违法事实详情 ID |
| `T006` | 扩展字段 |
| `T009` | 立案类型 |

`dbmx` 担保明细常用字段：`N001` 报告期、`N002` 担保方、`N003` 被担保方、`N004` 担保金额、`N005` 币种、`N008` 担保类型、`N009` 是否履行完毕、`N010` 是否关联交易、`N011` 实际发生日、`N012` 担保期限。

拿到详情记录 ID 后，可以用 [F10-详情正文.md](F10-详情正文.md) 查询正文。
