# 7709-分时副图

## 作用

查询个股分时页下方副图序列，例如买卖力道、成交对比。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.minutes.aux()` |
| 底层接口 | [`0x051b`](../COMMANDS_7709.md#cmd-0x051b) |
| 返回模型 | `MinuteAuxSeries` |

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    strength = client.minutes.aux("sz000001", kind="buy_sell_strength")
    volume_cmp = client.minutes.aux("sz000001", kind="volume_comparison")
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `kind` | 副图类型，也可以传底层 selector 数字 |
| `include_raw` | 是否保留原始 payload |

| `kind` | selector | 含义 |
| --- | ---:| --- |
| `buy_sell_strength` / `buy_sell` / `commission` | `0x00` | 买卖力道 / 委买委卖口径 |
| `volume_comparison` / `volume_compare` | `0x0b` | 成交对比口径 |

## 解析字段

| `MinuteAuxSeries` 字段 | 含义 |
| --- | --- |
| `exchange` / `market_id` / `code` / `full_code` | 市场和代码 |
| `kind` | 副图类型 |
| `selector_raw` | 副图选择器原始值 |
| `points` | 副图点列表 |
| `count` | 点数 |
| `raw_payload` | 原始 payload |

| `MinuteAuxPoint` 字段 | 含义 |
| --- | --- |
| `index` | 分时序号 |
| `time_label` | 时间 |
| `series_a` / `series_b` | 两条序列的通用值 |
| `buy_commission` / `sell_commission` | 买卖力道口径字段，`selector=0x00` 时使用 |
| `previous_day_cumulative_volume` | 前一交易日累计成交量口径字段，`selector=0x0b` 时使用 |
| `current_day_cumulative_volume` | 当前交易日累计成交量口径字段，`selector=0x0b` 时使用 |
| `cumulative_volume` | 当前序列累计成交量，当前等于 `current_day_cumulative_volume` |
| `record_hex` | 单点原始十六进制 |

## 使用建议

`series_a` / `series_b` 是统一字段，适合画双序列图。已知 `kind` 时优先使用对应的语义字段，例如 `buy_commission`、`sell_commission`。
