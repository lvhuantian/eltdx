# F10-公司概况

## 作用

查询公司概况。默认 `section="8"` 返回发行上市信息。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.company_profile(code, section="8")` |
| 底层 Entry | [`CWServ.tdxf10_gg_gsgk`](../F10_7615.md#entry-cwserv-tdxf10-gg-gsgk) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
response = f10.company_profile("000034")

print(response.rows[:3])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `section` | 页面分区；`8` 常用于发行上市信息，`9` 常用于指数调入调出 |

## 请求对照

```python
f10.company_profile("000034")
```

实际请求：

```python
Entry = "CWServ.tdxf10_gg_gsgk"
body = {"Params": ["8", "000034", ""]}
```

## 字段对照

`section="8"` 返回发行上市信息，常用字段如下：

| 原生字段 | 中文含义 |
| --- | --- |
| `T035` | 股票类别 |
| `T031` | 上市日期 |
| `T051` | 发行方式 |
| `fxzd` | 发行制度 |
| `mgmz` | 每股面值 |
| `T042` | 发行价格 |
| `T044` | 总发行数量 |
| `T056` | 实际募资总额 |
| `T057` | 实际募资净额 |
| `T059` | 首日开盘价 |
| `T060` | 首日收盘价 |
| `zcxs` | 主承销商 |
| `bjr` | 上市保荐人 |

`section="9"` 返回指数调入调出信息，常用字段是 `N001` 公布日期、`N002` 方向、`N003` 指数名称、`N004` 公布日涨跌幅、`N005` 调整日期、`N006` 调整日涨跌幅。
