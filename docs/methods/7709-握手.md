# 7709-握手

## 作用

建立 7709 连接后的握手请求，返回服务端时间、交易时段、主站名和产品标识。普通业务请求会在真实 socket 首次请求前自动握手，通常不用手动调用。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.session.handshake()` |
| 底层接口 | [`0x000d`](../COMMANDS_7709.md#cmd-0x000d) |
| 返回模型 | `HandshakeInfo` |

## 示例

```python
from eltdx import TdxClient

with TdxClient(timeout=3) as client:
    info = client.session.handshake()
    print(info.server_datetime)
    print(info.server_name)
```

## 参数

| 参数 | 含义 |
| --- | --- |
| 无 | 不需要业务参数 |

## 解析字段

| `HandshakeInfo` 字段 | 含义 |
| --- | --- |
| `server_datetime` | 服务端日期时间 |
| `session_minutes_1` | 第一组交易时段分钟信息 |
| `session_minutes_2` | 第二组交易时段分钟信息 |
| `server_date_1` / `server_date_2` | 服务端日期 |
| `server_name` | 主站名称 |
| `product_tag` | 产品标识 |
| `unknown_time_1_raw` / `unknown_time_2_raw` | 原始时间相关字段 |
| `flags_raw` | 原始标志字段，保留 |
| `tail_control_raw` | 尾部控制字段，保留 |
| `raw_payload` | 原始 payload |

## 使用说明

| 项目 | 说明 |
| --- | --- |
| 是否要手动调 | 一般不需要；真实 socket 首次业务请求前会自动握手 |
| 什么时候手动调 | 想检查连接主站、服务端时间、交易时段时可以调 |
| 是否鉴权 | 当前 7709 行情接口没有额外账号鉴权流程 |
| 和心跳关系 | 握手是连接初始化；心跳是连接保持 |
