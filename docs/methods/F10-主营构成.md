# F10-主营构成

## 作用

查询主营业务构成，包括主营收入、成本、毛利、收入占比、毛利率等。

| 项目 | 内容 |
| --- | --- |
| 调用方法 | `client.f10.business_composition(code, report_date=None)` |
| 底层 Entry | [`CWServ.tdxf10_gg_jyfx`](../F10_7615.md#entry-cwserv-tdxf10-gg-jyfx) |
| 返回模型 | `F10Response` |

## 示例

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
latest = f10.business_composition("000034")
fixed = f10.business_composition("000034", report_date="20251231")

print(latest.rows[:5])
print(fixed.rows[:5])
```

## 参数

| 参数 | 含义 |
| --- | --- |
| `code` | 证券代码 |
| `report_date` | 报告期，如 `20251231`；不传时先取服务端给出的最新报告期 |

## 请求对照

不传报告期时会先请求：

```python
Entry = "CWServ.tdxf10_gg_comreq"
body = {"Params": ["zygcfx", "000034"]}
```

拿到报告期后再请求：

```python
Entry = "CWServ.tdxf10_gg_jyfx"
body = {"Params": ["000034", "zygc", "20251231"]}
```

## 字段对照

`response.rows` 默认取第一张表。第一张表通常按行业、产品或地区分行返回主营构成：

| 原生字段 | 中文含义 |
| --- | --- |
| `N000` | 分类方式 |
| `N001` | 序号 / 层级 |
| `N002` | 主营构成名称 |
| `N003` | 主营收入 |
| `N004` | 收入比例% |
| `N005` | 主营成本 |
| `N006` | 成本比例% |
| `N007` | 毛利 |
| `N008` | 利润比例% |
| `N009` | 毛利率% |

有些响应还会带元信息表，常见字段是 `rq` 报告期、`nhytype` 口径类型、`ct_acp` 统计口径参数。需要看所有表时用：

```python
for table in response.tables:
    print(table.key, table.columns, table.rows[:2])
```
