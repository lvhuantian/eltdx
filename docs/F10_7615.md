# 7615 F10 / TQLEX API

`7615` 是通达信 F10 / 资料数据的 HTTP 网关，适合查询公司概况、题材、公告、财务报表、估值、研报等资料类数据。

它和 `7709` 行情主站是两条线：

| 类型 | 7709 | 7615 |
| --- | --- | --- |
| 连接方式 | TCP 长连接 | HTTP POST |
| 数据类型 | 行情、K 线、分时、成交明细、代码表 | F10、财务、题材、新闻公告、研报 |
| 使用入口 | `TdxClient` | `client.f10` 或 `F10Client` |
| 是否需要 7709 握手 | 需要 | 不需要 |

## 基本用法

```python
from eltdx import TdxClient

client = TdxClient(timeout=3)
profile = client.f10.company_profile("000034")
topics = client.f10.hot_topics("000034")
notices = client.f10.announcements("000034")

print(profile.rows[0])
print(topics.rows[:3])
print(notices.rows[:3])
```

也可以只用 F10 HTTP 客户端：

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)
print(f10.company_profile("000034").rows[0])
```

## 返回结构

所有 F10 方法统一返回 `F10Response`：

| 字段 / 属性 | 含义 |
| --- | --- |
| `entry` | 实际调用的 TQLEX Entry |
| `request_body` | 实际发送的 JSON body |
| `error_code` | 服务端 ErrorCode |
| `ok` | `ErrorCode` 为 `0` 或空时为 `True` |
| `tables` / `result_sets` | 返回的表 |
| `rows` | 第一张表的行，适合最常见的单表结果 |
| `raw` | 原始 JSON 响应 |

每张表是 `F10ResultSet`：

| 字段 / 属性 | 含义 |
| --- | --- |
| `key` | ResultSetKey，没有时自动用 `table0`、`table1` |
| `columns` | 原生列名 |
| `rows` | 字典行 |
| `row_cells` | 带列位置的单元格，适合处理重复列名 |
| `count` | 行数 |

如果服务端返回重复字段名，`rows` 会自动保留后续列：

```text
T120, T120 -> row["T120"], row["T120__2"]
```

F10 字段以服务端返回的 `ColName` / `ColDes` 为准，很多 Entry 会返回 `T003`、`N001` 这类原生列名。单个调用方法的常用字段对照见 [methods/README.md](methods/README.md) 里的 F10 单页；遇到未列出的字段时，先看：

```python
print(response.tables[0].columns)
print(response.tables[0].rows[:2])
```

## 通用 Entry 调用

没有封装到方法里的 Entry，也可以直接调用：

```python
from eltdx import F10Client

f10 = F10Client(timeout=3)

response = f10.call(
    "CWServ.tdxf10_gg_gsgk",
    params=["8", "000034", ""],
)

print(response.rows[0])
```

对于 `HQServ.*` 这类使用自定义请求体的 Entry，直接传完整 body：

```python
response = f10.call(
    "HQServ.hq_nlp_tcihq",
    [{
        "ReqId": "200743",
        "setcode": 0,
        "code": "000034",
        "Page": -1,
        "PageSize": "-1",
        "modname": "mod_tcihq.dll",
    }],
)
```
